# managustin.vercel.app

Personal portfolio built with Vite, Tailwind CSS v4, and vanilla JavaScript. Live at **[managustin.vercel.app](https://managustin.vercel.app)**.

## About

Computer Science Engineering student from Argentina. Interested in backend development, networking, and cybersecurity. Currently studying for the CCNA.

## Features

- Bilingual (EN / ES) with a custom i18n system
- Light/dark mode that transitions on scroll via IntersectionObserver
- Animated background canvas
- Contact form powered by EmailJS
- Hardened security headers deployed via Vercel ([full audit report](./SECURITY_REPORT.md))
